# Micahcraft

Micahcraft is my personal Minecraft server datapack (and associated build tooling). This repo also
contains the server’s resource pack sources.

## Licensing

This repository uses **path-based licensing**.

Canonical license texts live in `/LICENSES/`.

- `/LICENSE` — MIT (default for the repository, including `/src/**` and tooling)
- `resources/assets/**` — CC BY-NC 4.0 by default (see `/LICENSES/CC-BY-NC-4.0.txt`)
  - `resources/assets/micahcraft/textures/painting/jessica_kohler/**` — All Rights Reserved (see `/LICENSES/ARR-Jessica_Kohler.txt`)
  - `resources/assets/micahcraft/textures/estecka/**` — CC BY-NC-SA 4.0 (see `/LICENSES/CC-BY-NC-SA-4.0.txt`)

Folder-level pointer notices are included alongside assets to make the applicable license obvious when browsing the tree.
